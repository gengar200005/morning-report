## 1. 배경

실전 운용 관찰:

> 리포트의 RS Top 5 가 이미 신호일 10~18일차 종목들이 대다수라 Extended/Climax
> 구간 진입이 빈번하다. 백테의 "신호 1일차 종가 진입" 가정과 불일치해 백테
> 우위 (+29.55% CAGR) 가 재현되지 않을 수 있다.

이 가설을 실증하기 위해 세 실험을 돌림:

- **실험 A**: 현 `strategy.run_backtest` 의 진입 경로 추적 + 각 trade 의
  `signal_age_at_sel` (선정 시점 연속 신호 streak) 분포 측정
- **실험 B**: 필터·체결 조합 4 variant 분리 (수급·RS_min 유무 × 익일시가·당일종가)
- **실험 C**: `signal_days ≤ N` 강제 필터 민감도 (N=1/2/3/5/10)

구현: `backtest/experiments/` (기존 `backtest/strategy.py` unchanged).

## 2. 결과 요약

### 2.1 실험 A — baseline 재현 + signal_age 분포

| 항목 | 값 |
|---|---:|
| CAGR | **+29.55%** (기존 T10/CD60 재현) |
| MDD | -29.83% |
| Trades | 333건 |
| 승률 | 42.3% |
| PF | 2.21 |
| 평균 보유일 | 26.8일 |

**signal_age 분포 (선정일 기준 연속 신호 streak)**:

| 구간 | 건수 | 비율 |
|---|---:|---:|
| 1일 (fresh) | 65 | **19.5%** |
| 2-3일 | 41 | 12.3% |
| 4-7일 | 76 | 22.8% |
| 8-14일 | 74 | 22.2% |
| 15-30일 | 61 | 18.3% |
| 31-60일 | 15 | 4.5% |
| 61+일 | 1 | 0.3% |

- median **6일**, mean 9.7일, p75 = 13일, p90 = 23일
- 기간별 median 은 박스권/중립/강세 모두 6일로 일관
- **fresh signal (1일차) 진입은 전체의 1/5 뿐**. 나머지 80.5% 는 "extended" 상태

→ 가설 1 (백테는 1일차 종가 진입) 은 **오류**. 백테도 실전과 동일하게 extended
종목을 다수 잡고 있으며, 실전 괴리의 원인이 아님.

### 2.2 실험 B — 필터 vs 체결 기여도 분리

| Variant | 필터 | 체결 | CAGR | MDD | 승률 | PF | Δ CAGR |
|---|---|---|---:|---:|---:|---:|---:|
| B0 baseline | check_signal | open_next_day | **+29.55%** | -29.83% | 42.3% | 2.21 | — |
| B1 | check_signal | close_same_day | +22.85% | -32.14% | 39.8% | 1.96 | **-6.70%p** |
| B2 | core_only | open_next_day | +3.74% | -39.16% | 29.5% | 1.18 | **-25.81%p** |
| B3 | core_only | close_same_day | +13.30% | -38.84% | 32.1% | 1.52 | -16.25%p |

**해석**:
- **체결 타이밍 (당일종가) 영향 = -6.7%p CAGR**. 의미있지만 지배적이지 않음.
- **필터 완화 (수급·RS_min 제거) 영향 = -25.8%p CAGR**. 알파의 대부분이 여기서 옴.
- B3 가 B2 보다 역설적으로 나은 이유는 추가 분석 필요 (후속 과제).

→ 알파의 본질은 **"check_signal = Minervini + 수급 + RS≥70 필터가 걸러내는
지속 종목"**. "extended 진입" 자체는 알파 파괴 요인이 아님.

### 2.3 실험 C — signal_days ≤ N 필터 기각

| max_streak | CAGR | MDD | 승률 | PF | Δ CAGR |
|---:|---:|---:|---:|---:|---:|
| baseline (무필터) | **+29.55%** | -29.83% | 42.3% | 2.21 | — |
| ≤ 1 (fresh only) | +18.72% | -41.96% | 35.5% | 1.79 | **-10.83%p** |
| ≤ 2 | +22.54% | -41.01% | 39.0% | 1.97 | -7.01%p |
| ≤ 3 | +18.75% | -54.30% | 36.3% | 1.79 | -10.80%p |
| ≤ 5 | +25.84% | -43.21% | 38.1% | 2.15 | -3.71%p |
| ≤ 10 | +25.24% | -31.48% | 40.8% | 2.09 | -4.31%p |

**모든 변형이 baseline 하회**. fresh 로 강제할수록 MDD 가 악화 (-31.5% → -54.3%).

→ **"추세 지속성" 이 알파의 일부**. 첫 신호는 거짓 돌파 비율이 높고, 며칠 견딘
신호가 진짜. 사용자의 "extended 진입 우려" 는 실증적으로 **기각**.

## 3. 검토한 대안

### 3.1 signal_days ≤ 3 필터 채택 (프롬프트 제안)
- 실험 C 결과 CAGR -10.8%p, MDD -24.5%p 악화 → **기각**.

### 3.2 signal_days ≤ 10 필터 채택 (MDD 보호 목적)
- MDD 는 baseline 수준 (-31.5%). CAGR -4.3%p 감소.
- tradeoff 존재하나, 2025+ 구간에서 +183% vs baseline +161% 로 오히려 높음 → 재검토 여지 있음.
- 지금은 **보류** (추가 실험 필요).

### 3.3 체결 타이밍을 실전 동형(당일종가) 으로 이관
- B1 결과 CAGR -6.7%p → 실전 기댓값은 **+22.85% 기준**으로 하향 조정 필요.
- 백테 CAGR +29.55% 는 익일 시가 전제 하의 수치.
- 실전 "리포트 직후 당일 종가 매수" 방식이면 -6~7%p 추가 차감 반영.

### 3.4 baseline 유지 (수급+RS_min+check_signal 그대로)
- **채택**. 알파 검증 완료.

## 4. 결정

### 4.1 전략 파라미터 — 변경 없음
- `strategy_config.yaml::T10/CD60` 그대로 유지.
- `check_signal` (Minervini core + 수급 + RS≥70) 가 알파 원천이므로 보존.

### 4.2 실전 기댓값 보정
실험 B1 (체결 타이밍 당일종가) 기준으로 **실전 기댓값 재산정**:

| 시장 환경 | 백테 (B0) | 백테 (B1, 체결 현실화) | 실전 차감 후 |
|---|---:|---:|---:|
| 박스권 (2015-19) | -2.05% | 보정 필요 | -5 ~ +3% |
| 중립 (2020-24) | +35.26% | 보정 필요 | +25-30% → **+18-24%** |
| 강세장 (2025+) | +157% | 보정 필요 | 판단 보류 |
| **전체** | **+29.55%** | **+22.85%** | **+15-20%** (기존 유지) |

→ 기존 CLAUDE.md 의 "실전 기댓값 +15-20%" 는 **이 체결 현실화 손실을 이미
반영한 값이므로 변경 불필요**. B1 의 +22.85% 에서 다시 -3~7%p (슬리피지·
세금·생존편향) 차감하면 기존 기댓값과 일치.

### 4.3 Extended 진입에 대한 운영 규약
- "리포트 Top 5 가 signal_age 10~18 이어도 불건전한 상태 아님". baseline 이
  같은 분포를 보이며 알파 발생시킴.
- 따라서 **"신호 N일 이내 종목만 매수" 같은 자체 규칙 도입 금지**. 백테가
  그 제약을 주입했을 때 모두 악화.
- 단, 개별 종목의 **Climax top (상승 가속 후 매도 거래량 폭증)** 은 다른 차원의
  이슈 — 별도 ADR 후보로 분리 (volume-climax 감지).

### 4.4 파기/대체 — VCP 알고리즘화 계획
현 실험 결과는 "필터 강화가 알파" 를 지지. VCP 알고리즘화는 **필터 추가 방향**이
되어야 한다:
- 현 필터: Minervini 8조건 + 수급 20일 + RS≥70
- VCP 추가 후보: 피벗 고점, 수축 비율, 돌파 거래량
- **선후관계 재조정**: fresh-signal 필터 도입 계획은 **폐기**. VCP 는 "지속
  신호 중에서 수축 패턴 보이는 종목" 우선 선정 로직으로 재설계.

## 5. 후속 과제

- [ ] **B2 vs B3 역전 원인 분석** — 필터 완화 + 당일종가가 필터 완화 + 익일시가
      보다 나은 이유. 샘플 trade 레벨 분석 필요. (우선순위 중)
- [ ] **체결 타이밍 공식 입장 확정** — 백테 기본값을 open_next_day 로 유지하되,
      실전 모드 대비 수치를 `strategy_config.yaml::execution.entry` 옵션으로
      노출하고 -6.7%p 손실을 문서에 명시. (우선순위 상)
- [ ] **Climax top 감지** — signal_age ≥ 30 & 최근 5일 거래량 급등 구간에서만
      exit 가속 하는 규칙 실험. (ADR-006 후보)
- [ ] **signal_days ≤ 10 + MDD 개선 검증** — 2025+ 강세장 아웃퍼폼이 우연 인지
      규명. walkforward 분해 필요.

## 7. 교훈

1. **"Extended 진입이 실전 괴리 원인" 가설은 직관적이나 실증 기각됨**. 백테도
   같은 패턴을 보이며 알파 유지. 직관 → 실증 루틴이 없으면 잘못된 개선 방향으로
   이끌 뻔함.
2. **"fresh = 좋다" 의 심리적 편향**. 신규 종목을 잡는 편이 "알파답다" 는 직관은
   수치상 성립 안 함. 첫 신호의 가짜 돌파 비율이 높다는 반증.
3. **2-변수 동시 변경의 위험** — 실험 B 초안은 필터·체결 둘 다 바꿔 해석 불가.
   분리 실험 (B1/B2/B3) 없으면 "extended 가 문제" 같은 잘못된 결론 유도 가능.
4. **알파 원천 분해** — 백테 +29.55% 는 (체결 타이밍 알파 +6.7%p) + (필터 알파
   +25.8%p) 구조. 실전화 시 체결 알파 대부분 소실되고 필터 알파만 남음. 실전
   기댓값 +15-20% 는 이 분해와 일치.
