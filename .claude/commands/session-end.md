오늘 세션 마무리. 요약: $ARGUMENTS

순서대로:

1. **`SESSION_LOG.md` 업데이트**
   - 최상단에 오늘 날짜(YYYY-MM-DD) 섹션 추가
   - 4가지 항목 채우기:
     - `### 결정`: 이번 세션에서 내린 결정
     - `### 검토한 대안`: 고려했지만 선택 안 한 옵션
     - `### 다음 세션에서 할 일`
     - `### 미해결`: 답 못 찾은 이슈

2. **`CLAUDE.md` 업데이트**
   - 맨 위의 `<!-- Last session branch: ... -->` 포인터를 **현재 브랜치로 갱신**
     (`git branch --show-current` 결과 + 오늘 날짜)
   - "현재 상태" 섹션을 오늘 기준으로 덮어쓰기
   - "최근 세션" 목록에 오늘 날짜 + 한 줄 요약 추가 (최대 5개 유지)
   - "활성 작업 - 다음 단계 후보"도 진행 상황에 맞게 조정

3. **ADR 필요성 체크**
   - 오늘 내린 결정 중 "되돌리면 파장이 큰 구조적 결정"이 있으면 알려줘
   - 있으면 ADR 파일 만들지 물어봐 (자동 생성 금지)

4. **커밋 제안**
   - 변경사항 커밋 메시지 제안: `docs: session YYYY-MM-DD - [한 줄 요약]`
   - 승인하면 커밋, 아니면 대기

⚠️ 코드 파일은 건드리지 마. 문서 파일만 업데이트.
⚠️ git push 절대 자동 금지 (마스터 명시 승인 규칙).
⚠️ 브랜치 포인터 갱신은 필수 — 안 하면 다음 세션이 또 누락 위험.
